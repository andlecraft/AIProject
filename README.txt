당신의 신뢰도는? 플러그인을 방문해 주셔서 대단히 감사합니다. 

-저희 플러그인은 아직 개발 단계에 있습니다.- 

사용법 

1. 크롬 오른쪽위 점3개 
2. 확장프로그램 -> 확장프로그램 관리 
3. 오른쪽 위 개발자 모드 활성화 
4. 저희의 project/Extension 파일을 압축 해제된 확장프로그램을 로드합니다. 버튼을 통해 올려주세요 ! 

개발일정 

마지막 개발 커밋 이후로 잠시 기획 수정/정보 수집 기간을 갖고 있습니다. 
아직 주요 기능들 (예: ai , 서버 등) 이 구현되지 않은 상태이며 곧 다시 개발에 착수할 예정입니다. 
기대해 주시면 감사하겠습니다.
